nativos@cloud:~/produtos/vanishd
Security
vanishd

Plataforma de compartilhamento de segredos com zero-knowledge. O servidor nunca vê o conteúdo — toda a cifragem acontece no cliente, e os links se autodestruem após o primeiro acesso.

  • AES-256-GCM client-side via Web Crypto API
  • Chave de decifragem no fragmento da URL (invisível ao servidor)
  • Proteção por senha com PBKDF2 e deleção atômica na primeira leitura
  • Rate limiting por IP e headers de segurança com CSP

mfa-app

Plataforma web centralizada para gestão de tokens TOTP em times. Geração de códigos em tempo real com timer de 30 segundos e modo demo para onboarding rápido.

  • Registro de tokens via leitura de QR Code
  • Modo de edição protegido por senha
  • Atualizações de código TOTP ao vivo com indicadores de progresso
  • Deploy imediato com SQLite ou MySQL via Docker Compose
